¿QUIÉNES SOMOS?

Somos un grupo de tres estudiantes del ciclo superior de Desarrollo de Aplicaciones Web (DAW2) en el instituto Bernat el Ferrer.

¿CÓMO SURGIÓ?

Este proyecto fue desarrollado como parte de nuestro curso, buscando ofrecer una solución tecnológica a un problema común: la gestión de plazas de parking.

¿QUÉ ES?

Se trata de una aplicación web que permite consultar la ocupación de parkings en tiempo real, gestionar reservas y ofrecer información detallada.
